Directories:
- `lib` - source code for a library used by `main` and that can also be used to create customized derivatives of `francis`
- `main` - the actual `francis` executable
- `demo` - source code for `francis-demo`, which demonstrates how to customize a derivative of `francis`
- `demo-app` - an app bundled with `francis-demo`
- `demo-instrumentation` - a benchmark bundled with with `francis-demo`
- `scripts` - scripts for build/running/testing francis
- `prebuilt` - stuff copied into the release verbatim
